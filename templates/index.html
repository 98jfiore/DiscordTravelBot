TravelBot

What is it?

This Bot is a chatbot for a discord server. It acts as a game where you can travel around the world country by country. Earn miles by participating in servers with TravelBot and talk to TravelBot to get info about where you are or take a flight to the next country over.

What was used?

This app was developed in python with the discord.py library used to interface with discord servers. The API https://restcountries.eu/ was used to get country information, and https://www.countryflags.io/ is used to get images of country flags. PostgreSQL was used to store information. The app was also launched on Heroku and posted on GitHub.

If you own a Discord server, simply click on the link below, log in to Discord, and select the server you wish to add the bot to.
